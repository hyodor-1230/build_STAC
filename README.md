\# STAC API Server



위성영상 메타데이터 검색을 위한 STAC 기반 API 서버 구축 프로젝트입니다.



\## 프로젝트 개요



본 프로젝트는 STAC SpatioTemporal Asset Catalog 구조를 기반으로 위성영상 데이터를 조회할 수 있는 API 서버를 구축한 프로젝트입니다.



\## 주요 기능



\- STAC API 서버 구축

\- 위성영상 메타데이터 조회

\- Python 기반 API 실행

\- AWS EC2 환경 배포



\## 기술 스택



\- Python

\- Flask 또는 FastAPI

\- AWS EC2

\- Git / GitHub

\- STAC
