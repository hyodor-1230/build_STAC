# STAC API Server

STAC 기반 위성영상 데이터 조회 및 다운로드를 위한 API 서버 구축 프로젝트입니다.

사용자는 별도의 Python 코드 작성 없이 웹 주소(URL) 입력만으로 원하는 날짜 또는 날짜 범위의 STAC 데이터 파일을 조회하고, `bbox` 값을 제공하면 관심 영역으로 클리핑된 결과를 다운로드할 수 있습니다.

---

## 주요 기능

| 기능 | 설명 |
|---|---|
| 특정 날짜 데이터 조회 | 입력한 날짜에 해당하는 STAC 데이터의 실제 파일 경로 반환 |
| 날짜 범위 데이터 조회 | `start`–`end` 기간에 포함되는 STAC 데이터 파일 목록 반환 |
| BBOX 기반 클리핑 | `bbox` 입력 시 관심 영역으로 데이터를 클리핑 |
| 결과 다운로드 | 클리핑된 결과 파일 다운로드 지원 |
| URL 기반 사용 | 별도 코드 작성 없이 웹 주소 입력만으로 사용 가능 |

---

## 사용 예시

### 특정 날짜의 데이터 경로 조회

```text
http://서버주소/endpoint?date=2024-01-01

### 특정 날짜의 데이터 경로 조회
http://서버주소/endpoint?start=2024-01-01&end=2024-01-31

### http://서버주소/endpoint?start=2024-01-01&end=2024-01-31&bbox=minX,minY,maxX,maxY

### 기술 스택
Python
STAC
AWS EC2
AWS S3
Git / GitHub

### 프로젝트 특징
STAC 메타데이터 기반 위성영상 데이터 탐색 자동화
날짜 조건 기반 데이터 검색 기능 구현
날짜 범위 기반 파일 목록 조회 기능 구현
bbox 기반 관심 영역 클리핑 및 다운로드 지원
AWS EC2 환경에서 API 서버 실행
사용자가 코드 작성 없이 URL 입력만으로 데이터 접근 가능
